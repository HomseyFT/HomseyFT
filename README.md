# Main Projects:

- **QuinLang**
  Custom compiler with a lexer, parser, semantic analysis, and codegen. Compiles into a custom bytecode, which is then interpreted by a VM, also written by me.
  [QuinLang](https://github.com/HomseyFT/QuinLang)
- **Homelab**
  Personal homelab hosted using Docker, exposed to the internet through a Cloudflare tunnel. Has a media server, network-wide adblocker/VPN, uptime monitoring, and an AI       agent, among other things.
  [Homelab](https://github.com/HomseyFT/Docker-Stack)
- **Image Compressor**
  Custom image compressor, currently only works on black and white images. Uses JPG-like compression (discrete cosine transform, quantization matrix, Huffman encoding).
  (Image Compressor)[https://github.com/HomseyFT/Image-Compression-Project]
